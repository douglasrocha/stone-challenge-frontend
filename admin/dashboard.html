Bem vindo ao admin panel do stone blog

O stone blog foi criado como desafio da Stone, e desenvolvido por mim em pouco tempo. Espera-se que o resultado deste software consiga demonstrar minha capacidade de resolver problemas e minha disciplina de finalizar tarefas e me preocupar com detalhes.

Este software foi criado usando algumas tecnologias de ponta e usando alguns conceitos padrões de mercado hoje. Segue abaixo as tecnologias utilizadas.

  • Visual Studio 2013
  • C# + Web API 2
  • Entity Framework (Code First)
  • Newtonsoft.JSON
  • Ninject
  • Microsoft SQL Server
  • Domain Driven Design
  • RESTFUL API
  • HTML5 + JQuery + Bootstrap
  • AngularJS

Finalmente, seja bem vindo ao painél de administrador. Selecione no menu a seção que deseja administrar e modifique os dados presentes no blog.
